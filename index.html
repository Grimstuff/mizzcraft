Mod Installation Guide

Mizzcraft recommends the following mods:

  1. Java Runtime Environment: Required to run Minecraft and mods. Download
  2. Iris Shaders: Beautiful lighting effects. (Select the 'Iris + Fabric' option during installation). Download
  3. Simple Voice Mod: Talk with friends in-game with proximity voice. Download

How to Install:

  1. Install Java, then Iris Shaders, select Iris + Fabric as the installation option.
  2. Open the Minecraft Launcher, browse to Installations, and for Fabric Loader click Open Installation Folder.


  3. Drop BSLClassic_v8.4.02.2.zip into the shaderpacks folder.
  4. Drop voicechat-fabric-1.21.4-2.5.28.jar into the mods folder (create the folder if none exists).
  5. Important: Make sure to select and launch Fabric Loader in the Minecraft launcher dropdown menu.

Alternative Easy Installer

Don't want to install mods manually?

  1. Download and install CurseForge App
  2. Open the app and click "Minecraft"
  3. Search for "Mizzcraft" in the search bar
  4. Click "Install" and wait for the download to complete
  5. Click "Play" and enjoy!
